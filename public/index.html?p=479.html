[收集]不止是BootStrap 分享简洁好用的前端框架

Bootstrap是目前最受欢迎的前端框架，由Twitter开发。Bootstrap是基于HTML、CSS、JAVASCRIPT的，它简洁灵活，使得Web开发更加快捷。

BootStrap固然很好，面对不同的开发需求，前端工作人员可能需要各种不同的架构来实现开发。博主就将自己见过不错的前端架构给大家收集汇总一下，抛砖引玉，有更好的推荐欢迎评论区留言。

BootStrap

首先是BootStrap，由Twitter开发，可能是目前应用最广泛的前端框架。

官方网站：https://getbootstrap.com/

中文网：http://www.bootcss.com/

成熟稳重，功能完善，包括CSS和JavaScript，功能较多。

有中文网站，帮助文档和示例都比较完善。

Ant Design

一个服务于企业级产品的设计体系，基于『确定』和『自然』的设计价值观和模块化的解决方案，让设计者专注于更好的用户体验。

官方地址：http://ant.design/

GitHub：https://github.com/ant-design/ant-design

这是一个国人的前端架构，昨天介绍的阿里内部最大文档平台“语雀”开放注册 免费使用使用的就是这个框架。

除此之外还有手机版

官方地址：https://mobile.ant.design

Mustard UI

只包含CSS文件的框架，压缩后体积仅为5.28KB。

官方地址：https://mustard-ui.com/

GitHub：https://github.com/kylelogue/mustard-ui

PURE

更加轻量化的CSS架构，压缩体积仅3.8KB。

官方在址：https://purecss.io/

做单页或简单项目绝对得心应手！

FrozenUI

FrozenUI是一套基于移动端的UI库轻量、精美、遵从手机 QQ 设计规范

官方地址：http://frozenui.github.io/

适用于使用手Q规范设计的Web页面，而针对非手Q规范的页面，可通过修改变量定制界面主题。

如果做移动端的页面，选择一款基于移动端的UI库，势必会事半功倍。

WeUI

WeUI 是一套同微信原生视觉体验一致的基础样式库，由微信官方设计团队为微信内网页和微信小程序量身设计，令用户的使用感知更加统一。

官方地址：https://weui.io/

仅适用于微信开发，界面比较朴素，提供原生微信视觉体验。

ZanUI

ZanUI-WeApp是有赞移动 Web UI 规范 ZanUI 的小程序现实版本，结合了微信的视觉规范，为用户提供更加统一的使用感受。

GitHub：https://github.com/youzan/zanui-weapp

适合微信小程序开发

总结

以上都是岩兔站实际工作中遇到比较好的框架，在此整理分享给大家，抛砖引玉。有更好的产品推荐欢迎下方留言。
