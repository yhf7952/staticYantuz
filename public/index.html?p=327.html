使用Coding部署WordPress

Coding 为您提供了一个基于 Linux 系统的虚拟机环境，包含完整的 MySQL 数据库功能，完美支持 WordPress、Typecho、OpenCart 等 PHP 应用。您无需购买服务器，无需备案，便能部署动态网站。

使用 Coding 部署动态网站，尽享 3 项超值服务：

  • 自定义域名
    可绑定自定义域名，无需备案，拥有一个真正属于自己的网站
  • 免费 SSL 证书
    自定义域名也能轻松享有 SSL 证书，全站 HTTPS 加密协议，远离被第三方拦截与伪造的风险
  • 一键部署
    真正一键部署，每一次更新代码仓库，网站自动更新为最新状态

查看成功部署的 示例 Pages 及 示例项目。

用户私有项目导入方式如下：

1、登录 Coding.net，新建一个私有项目，选择『导入仓库』，填入 WordPress 地址：git@github.com:WordPress/WordPress.git。

2、项目导入需要一点时间，请耐心等待片刻后刷新页面便可看到成功导入的代码文件。

3、在项目里点击『代码 -> Pages 服务』，选择『动态 Pages』 选项卡，点击『开启』，选择部署来源，点击『保存』。稍等片刻即可完成部署并通过 xx.coding.io 访问您的网站。

 

4、点击存储空间栏中的『连接信息』获取数据库连接信息。

注：数据库连接信息也可以通过环境变量方式获取，已经设置到了 $_ENV 里，分别是 MYSQL_DBNAME、MYSQL_USERNAME、MYSQL_PASSWORD、MYSQL_HOST、MYSQL_PORT。

5、打开部署的网站，填入第 4 步中获取的信息。

6、填写基本信息，安装 WordPress。

7、安装完成后，您的网站就搭建成功了！

使用第 6 步中的信息登录 WordPress 后即可进入管理后台。

8、在存储空间栏中点击『phpMySQLAdmin』，输入第 4 步获取的用户名与密码即可登录 phpMySQLAdmin。
