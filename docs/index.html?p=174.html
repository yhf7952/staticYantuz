随着区块链技术越来越火热，继招财猫项目之后，网易又推出了自己的区块链平台“网易星球”。网易星球是其于区块链的生态价值共享平台，用户可领取黑钻，黑钻总量有限，每两年产出量减少一半，随着时间的推移获取难度越来越大，黑钻是星球上的主要货币。

目前官方介绍还不多，项目也初建不久，本着早上车不吃亏的原则，小伙伴们还是早上车为好。

使用说明

网易星球目前是邀请注册制，所有注册用户必须通过朋友邀请才可注册，下面是博主邀请码，可扫码下载或在注册页输入邀请码“DQDL”完成注册。本邀请码还能用两次，如果用完了，新注册的用户请在页面下方留言。

星球系统主要分为黑钻和原力两个值，黑钻是星球的主要货币，也是玩家主要收集的东西。

 

原力值是促进黑钻上长的值，原力越大，则黑钻生长越快。获取原力值主要通过邀请好友，每日登录，或者暂未开放的独家任务获取。

新增的幸亏钻目前用途不明，估计也是提高黑钻产量的一种手段。
