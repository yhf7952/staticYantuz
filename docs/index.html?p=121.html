体验阿里云6元年付虚拟主机

2018-01-24 分类：资源分享 阅读(866) 评论(0)

文章目录

  • 购买链接

国外空间慢，国内空间贵，那有没有两全的解决方案呢？阿里云收购万网以来，俨然成为国内云服务器的龙头老大，很多人购买主机或空间都会首先阿里云。

今天给大家推荐的是阿里云最低端的一款虚拟主机，年付仅6元，还自带杀毒或日志分析。

购买链接

配置：200M网页空间/10G月流量/20M数据库

购买地址：https://wanwang.aliyun.com/hosting/free

使用体验

虚拟空间是自动开通的，支付6元后马上开通，系统会发邮件到注册邮箱，提示注册成功，以及注册之后的操作流程。

登录后台即可查看系统信息，PHP版本支持4.4-5.5，并不算丰富，而MySQL只支持5.1版本。

上传程序及数据库

如果是网站迁移，需分别备份网站，建议ZIP格式。和数据库，建议.sql形式。用FTP上传至空间后，可用后台自带的网站恢复工具进行恢复。其实就是分别解压程序，和执行sql脚本。

我的博客恢复完才发现，空间才40M占了20%，而数据库才仅0.48M。看来空间虽然紧凑，但是放下个人博客是完全不成问题的。

什么？200M空间装不下你的网站？可以用无比坚挺无比迅速的图床 微博图床，不但节省空间，还能节省80%的流量哦

访问速度

用惯了国外空间，再用国内空间的话简直不能更爽，由于空间是流量计费的，似乎并没有限制最大带宽，我的宽带上下行几乎都能达到满速。

关于备案

使用了国内空间就必定要备案，否则你遇到的就只有“网站暂不能访问”。关于备案，阿里云可能也算是做的最好的一个了，尽可能的简化了备案流程。

博主也在办备案，并打算把过程记录下来，给后来备案的站长一点经验。感兴趣的可以查看阿里云备案全记录
