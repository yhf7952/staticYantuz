博主使用的主题是DUX，用的久了，页面看起来很单调，因为全局都是灰蒙蒙的。如果换主题的话，工作量巨大，就想着写一些样式表为网站增加一些色彩点缀。

实际操作的过程中就是根本不知道用什么颜色，大红大绿的太扎眼，自己调个颜色，又说不上来的奇怪。看来作为一个没有审美的人来说，必须要求助于HTML配色表了。

官方网址

网上的配色表不在少数，这是我发现的界面最为清新，操作也最简单的了。

地址：https://htmlcolorcodes.com/

使用方法

现在流行扁平化，网站也提供了扁平化配色板。

选取颜色

点击一个色块显示其代码并将其添加到您的调色板。

保存颜色

在您的调色板上保存和撤销保存颜色；下次您访问时候它们会在这里。

导出颜色

将您当前的调色板以Hex、RGB、HTML、CSS和SCSS 格式输出。试一下吧！
