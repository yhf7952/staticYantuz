随着腾讯TIM的上线，腾讯文档也逐渐进入人们的视线，与TIM一同推出的就是在线文档编辑。由于腾讯TIM是定位于办公需求的，所以在线文档，多人协作也是必不可少的功能。而如今腾讯推出的腾讯文档则可脱离TIM，在网页、PC、手机、MAC等多终端无缝运行。

官方地址&简介

官方地址：https://docs.qq.com/

「腾讯文档」是什么？

腾讯文档是一款可多人协作的在线文档。

支持Word和Excel类型，打开网页就能查看和编辑，云端实时保存；

可多人实时编辑文档，权限安全可控。

打开网页就能写

无需下载安装，打开网址 docs.qq.com 立即开始编辑。

随时随地任您使用

PC、Mac、iPad、iOS和Android，任意设备皆可顺畅访问、创建和编辑文档。

无需特意保存，电脑断电文件不丢失

系统会对您的输入做自动保存，不用担心断网断电导致编辑的内容丢失，重新联网后文档内容自动恢复。

支持多人同时查看和编辑

多人协作无需反复收发文件，实时查看协作者的修改内容，并且可以查看修订记录。

强大的文档权限设置能力

查看和编辑权限云端可控，自主设置协作者的阅读和编辑权限，可针对QQ、微信好友设置文档访问权限。

实时翻译

自动识别语言，快速实现全文翻译，译文支持一键生成文档，方便保存和查看。

丰富的模板

支持工作日报、会议纪要、简历、工作日程和报名签到等模板。

在线版评测

进入https://docs.qq.com/desktop/即可看到腾讯文档平台中的所有在线文档。

文档导入

此处可以通过“新建”建立自己的在线文档，或“导入”现在已经编辑过的文档。

博主尝试导入几篇文档，发现纯图文形式的文档转换效果尚可接受，但是带表格的文档则是转换的乱七八糟。看来文档导入只适合导入最初级的文档，如果是直接在线创建表格，则不会出现变形问题。

文档工具栏

下面打开一篇示例文档，工具栏功能相对简单，包括“打印、格式刷、清除格式、字体字号、字体样式、颜色、序号、对齐、缩进、图像、表格”等功能。

其支持的只是最简单的文档功能，甚至比WordPress的文章编辑后台还要简陋。看来只能编辑最简单的文档，适合部分轻办公场合的需求。

水印和翻译

作为主打功能的水印和翻译功能做的十分便捷

一键翻译，翻译后的文档还能直接另存为新的在线文档，由于岩兔站英文也不好，所以翻译结果各位大佬自行评判。

一键加水印，简单快捷，比原生Word要方便的多，支持密集和宽松两种水印模式，并可对自己隐藏水印。

总结

优点：

  • 支持多终端
  • 在线使用，免安装
  • 适合多终端
  • 可多人同时编辑
  • 在线实时保存
  • 安全免费
  • 水印和翻译功能

缺点：

  • 转换文档能力较弱
  • 提供的文档编辑工具较少（不支持标题目录等）

总之，作为一款在线文档系统，虽然比不上微软OneDrive，但是对于国内用户来说，已经算的上是诚意之作了。免费且安全，适合团队协作文档的需求，是小型团队，或者轻办工商务人士的又一款在线文档利器。
