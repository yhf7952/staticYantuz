不知道各位亲的蛙儿子还好不，现在不养蛙改养狗了。莱茨狗是百度上线的一款区块链技术的狗狗集市，借助区块链技术，每一只狗狗都不可能是一样的。根据发型，肤色等和稀有程度，又分为了 “普通-稀有-卓越-史诗-神话-传说”6个等级。

首先打开官方地址：https://pet-chain.baidu.com/，使用百度帐号登录即可，登录之后即可免费领取一只狗。

如何领4只

https://pet-chain.baidu.com/chain/splash

https://pet-chain.baidu.com/chain/splash?appId=2&tpl=wallet

https://pet-chain.baidu.com/chain/splash?appId=3&tpl=wallet

https://pet-chain.baidu.com/chain/splash?appId=4&tpl=wallet

快速买狗

目前狗狗还不能繁殖，所以要想获得新狗只能通过买卖的方式了。如何通过现有的钱换到最多的狗，不二法门就是低买高卖了。百度在每比交易时还要收取5积分的交易税，所以交易前要提前计算好。

这里提供一个Chrome插件帮你快速买狗：

GitHub:https://github.com/tabalt/baidu-pet-chain-buyer

直接下载开源的源码，然后解压到本地。Chrome浏览器打开扩展管理，开启开发者模式，然后加载已解压的扩展程序即可

Chrome商店:https://chrome.google.com/webstore/detail/百度莱茨狗/fgpcgbppajchcfdnphppkhkfhbpnmoji

不能访问谷歌的同学可以通过https://chrome-extension-downloader.com/输入Chrome市场地址，进行下载，然后将crx文件拖入浏览器完成安装。

点击Chrome扩展，先设置买入价格

然后打开狗狗集市，开启调试模式可以看到购买日志，推荐按F12开启控制台，点击扩展“开始买入”

目前博主还没有成功买到低价狗，控制台报错网络异常，估计是现在玩狗的太多了，网络不畅导致的。

新版购买插件（2018-02-11 14:53:10更新）

点击插件，进入低价买狗页面，页面自动刷新，寻找符合要求的低价狗

一旦找到即刻弹出对话框，输入验证码即可购买成功

讨论

官方表示使用百度系产品，均可获得积分，这款产品很可能是借助养狗了推广自家产品，而每次交易的抽成，明显是为了让市场中的积分不断减少，而不得不向第三方渠道获取积分。

区块链最成功的是比特币，百度目前也来凑热闹，关于积分能不能用作其他交易，目前官方明确表态积分只能用于集市买狗并不能有其他用途。

关于繁殖，目前仍然没有繁殖功能，但是每只狗都标记了0代狗，就意味着迟早会支持繁殖的，到时候会有大量的新狗子产生。
