传图片有图床，比如新浪微博图床，那么传视频有没有免费安全的视频床呢？答案是肯定的！今天介绍一下openload.co一款免费无限空间的视频平台，支持在线播放，分享给别人查看还能获得佣金。

openload虽然是国外平台，但是国内访问速度还是不错的，上传下载基本能达到满速，官方也表示不限速。

官方地址

https://openload.co/

注意事项

在线演示播放的文件连续60天没有请求就被删除。其他格式的文件连续30天没有请求就被删除！

支持在线演示播放的格式有：

  • 音频: MP3;AAC;Ogg Opus;WebM Vorbis;WebM Opus;WAV-PCM
  • 视频: MP4; WebM; Ogg Theora
  • 文档: .html; .php; .txt

注册

右上角”Sign up today”（地址：https://openload.co/register）填写邮箱地址即可快速注册

上传/预览文件

首页或者https://openload.co/upload直接拖拽或选择文件上传，从我这里测试来看，确实是没有限速，达到了带宽最大上行

上传之后会自动生成链接，可以直接通过链接观看或下载。

分享或嵌入博客中

预览界面点击下方Embed或者进入文件管理后台均可得到iframe代码，直接嵌入文章中即可

开启赚钱模式

新注册用户默认没有开启赚钱模式，需要手动开启，打开https://openload.co/account#affiliatesettings点击同意开启。

开启之后需设置广告级别，以及是否强制关闭用户的广告屏蔽程序。

当然广告越多，用户每次观看获益越多，但体验会变差，至于怎么设置，需要看具体需求了。

赚钱规则

  1. 满20美元可提现，24小时内到帐
  2. 同IP观看和下载每天最多计3次
  3. 如果广告被广告屏蔽软件屏蔽了，那么所有收益将是原收益的10%
  4. 英国、美国、加拿大、澳大利亚，IP最值钱，每10000观看有$30，欧洲国家，南非，亚洲国家等费用递减，最低只有$4每万次观看。

其他特点

支持远程上传：https://openload.co/account#remoteupload

支持批量上传：https://openload.co/account#multiupload

支持FTP(https://openload.co/account#usersettings)和API(https://openload.co/api)

使用场景

博客或者公司可以使用其存放宣传片或者小视频等。

如果你愿意甚至可以建立自己的影视站。

老司机可能会觉得这个这个有点openload眼熟，因为他支持19+内容。
