#春联#新春到 腾讯AI人工智能写春联

2018-02-15 分类：资源分享 阅读(1475) 评论(0)

新春到，在这里先祝大家春节吉祥，狗年大吉。说到春节，最先联想到的就是春联了，随着人工智能的发展，现在AI也能帮我们写春联了。今天就介绍腾讯旗下的AI春联机器人。

使用方法

春联地址：http://couplet.ronghuiad.com（仅支持手机访问，电脑访问会显示二维码，微信扫描即可使用）

开始是一个漂亮的过场动画，动画完成之后展示的是以“腾讯”藏头的示例春联，点击下方“为我写春联”即可进入自定义春联界面。

自动生成春联可输入2-4字的中文以作为藏头，经博主实测，输入多字时，春联会取后两字做为藏头，所以使用时输入两字即可，比如“岩兔”。

点击“下一步”即可生成一副春联，如果不喜欢可以“换一换”。

有什么用途：生成的春联当然不能帖了，如果有会书法的老师，可以参考AI生成的春联，自己写一副个性化的春联。更多的是把它另存为图片，以品牌或者团队名称作为藏头，送给朋友和合作伙伴。
