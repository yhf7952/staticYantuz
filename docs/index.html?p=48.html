使用流程（以1美元虚拟主机为例）

第一步要申请域名，或者是用自己已有域名，这个域名也就是默认绑定空间的域名，所以最好是先确定域名之后再来购买空间。

点击使用按钮，检查通过后，进入商品详情页，默认是1美元年付，下边扩展服务不用勾选。

继续之后再次核对产品，确实无误就可以支付了。支付后产品会自动开通，同时注册邮箱会收到开通邮件，包括服务器IP地址，后台管理地址用户名及密码，FTP连接方式等信息。

然后通过邮件中的地址和用户名密码登录管理后台

管理后台也是支持中文的，包括所有功能在内，应有尽有
