年终总结写了没？教你写总结

2018-01-31 分类：资源分享 阅读(1362) 评论(0)

文章目录

  • 年终总结的意义
  • 年终总结写法

2017已经溜走，2018已经到来。也许很多小伙伴已经在元旦时做过了年终总结，但我相信，还有一帮苦逼上班族拖到了农历年才开始总结，而博主正在其中，那么怎么才能速成一篇年终总结呢，今天博主来分享下我的经验。

年终总结的意义

要想写好总结，总得闹明白什么是年终总结，又有什么存在的意义？

我就想了，这除了耽误大家的时间和精力，还能有什么意义呢？每个人都要写两页的总结，而领导还要看每个人的总结，所以两头都费劲，又是何必呢？

其实存在必有因，年终总结并不像想象中的那么鸡肋。除了概述一下一年的工作之外，还是一个向领导吐露心声的机会。而作为领导，当然更希望多的了解每一个员工的想法。

年终总结写法

开篇：

当然是时光荏苒岁月如棱，又是一年……除了抒情之外，也可以直接了当的写成“现将xxx2017年度工作总结如下”反正再多的抒情也都是空话，也不会有人介意的。

正文：

正文开始，就到了表达你一年工作的成果了。如果工作是多方面的，可以分多个段落依次阐述。而这个时候，如果能有数字说明，那就再好不过了，因为再多的语言都抵不过数字的严谨。

比如作为客服可以说“一年接打了3000通电话，解决了用户2500余次问题”。这样一来，工作量多少就一目了然了。而程序猿可以写一年新增了200个功能，修复了个100个bug等等。如果没做好记录，不知道自己一年做了多少呢？可以按周，或月预估一下工作量，再乘以12就能得出全年工作量。

大面积的工作列举完，如果篇幅不太够的话，可以找一两个典型工作，比如一次出差，一次接见大客户，一次成功的推销等等。在描写这件事情的时候顺便不露痕迹的夸奖了自己做的实在太棒了，当然，除了自己的努力，也是离不开领导的提拔和同事的协助。

结尾：

结尾通常是所有人都会写的地方。要放到结尾的内容是自我剖析，来年计划等等。但是千万别因为不会写就草草收尾，领导反而最看重的就是结尾，他会认为以上所有你汇报的内容，全是你份内之事，并没有什么亮眼之处。而真正让领导眼前一亮的，就要看结尾了。

首先，写写工作中的不足，别以为写了自己的不足会被留下不好的印象，正所谓人无完人，有错误的一年才算是完整的一年，当然这个错误只要不是渎职就行。然后讲讲自己是怎么解决的，或者是同事是如何帮称的，也对同事的工作做出肯定。

其次是新年展望，和工作计划。什么，你不知道怎么写？虽然各行种业的计划都不相同，但是有一条准则是肯定的，就是靠着你的前辈，上级的方向写准没错。比如要学会你主管的技能，比如希望超过优秀员工等等，总之给自己一个积极向上的能量。

最后就是真正的结尾了，这块实际意义和开头第一段一样，并没什么用处，无非是唱唱赞歌，打打鸡血，以一个升调结束整篇总结。
