网站流量多少才正常?用SimilarWeb分析大佬网站找差距

2018-09-02 分类：资源分享 阅读(165) 评论(0)

文章目录

  • SimilarWeb
    • 实现原理
  • 网页版
    • 大佬的网站
  • Chrome插件

网站做了很久了，从刚开始的无人问津，到流量逐渐增多。岩兔站开始和很多新手站长一样开始思考流量到多少才算成功，其实这根本是无解的一件事，流量没有最多，只有更多…站长能做的只能是自我超越，或者与大佬的网站对比。

自我越越比较好理解，这个月日均PV达到1000，次月达到1500就是超越。光是这么比较似乎也不知道到要到多少才算成功，如果能知道大佬们的网站有多少PV，似乎会更有奋斗的方向。

但是大家知道，只有安装网站统计代码才能精确的得知网站的精确访问量，要得知其他大佬网站的访问量几乎不可能。直到岩兔站发现SimilarWeb他能预估任何网站的月访问量，这样进行网站对比，找差距就非常方便了。

SimilarWeb

SimilarWeb可以查看某个网站的统计数据，比如访问量、访问来源、每个用户的浏览时长等等。

实现原理

岩兔站大体看了一下，因为英语水平有限，所以也没看明白他是怎么在不通过统计代码实现的访问量预估。猜想可能是根据搜索引擎搜索，或域名DNS解析请求等方面进行的统计。

网页版

通过网页可以直接搜索某网站的访问量情况，分析的项目可以参考岩兔站的分析结果。

官方地址：https://www.similarweb.com/

比较有用的分析结果包括：

  • 世界排名和国内排名
  • 月访问量，以及访问页面数量，停留时间，跳出率等
  • 用户分布地区（以国家区分）
  • 来源方式
  • 搜索词

下图是岩兔站的月度访问量分析，可见趋势是正确的，因为是做站时间还短，所以起初是0，后来慢慢长上来。

月度访问量就以7月为例，网站给出的统计结果是60K的访问量，岩兔站默默的打开百度统计，算了一下访问量，发现我的网站根本没那么多访问量……由此看来，SimilarWeb给出的结果跟实际结果还是有出入的。但是换一个角度来看，即使与实际值不同，但在他的统计算法下，各个网站应该都是公平的，所以仍然可以对比自己与大佬的差距。

大佬的网站

每个人心目中都有一个大佬，所以每个人的奋斗目标也不一样。

岩兔站崇拜的大佬有很多，其中阮一峰老师就是其中之一，他的每周分享都是必读的，来看一下阮老师博客的访问量怎么样。

真不愧是大佬，网站流量在4M-6M之间，比岩兔站对比多出两个数量级，真是惭愧啊。当然我的网站与阮老师的根本没法比，但是好不好比过才知道，由此可见努力的空间还是非常大的！

Chrome插件

如果想更简单方便的查看浏览到每一个网站的统计信息，建议安装一下SimilarWeb提供的Chrome插件，只要打开了网站，点击插件，即可快速查看当前网站的访问情况。插件使用起来会比网页版更方便，统计结果也比较集中、直观。

插件地址：https://chrome.google.com/webstore/detail/hoklmmgfnpapgjgcpechhaamimifchmp

无法访问应用市场的用户可前往https://chrome-extension-downloader.com/输入Chrome市场地址，进行下载，然后将crx文件拖入浏览器完成安装。
