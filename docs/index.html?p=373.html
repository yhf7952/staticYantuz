Chrome同步助手 借助插件实现国内环境同步浏览器数据

从易用性和市场占有率，Chrome浏览器目前都以绝对优势排行第一。但是因为你懂得原因，国内使用Chrome有诸多不便。比如Chrome最方便的书签、密码、访问记录、等同步功能均无法使用。Chrome同步助手这款插件就是为解决此问题而生，一个插件解决所有Chrome同步问题。

官方下载

官方地址：http://www.long-int.com/chrome-helper/

插件下载：右键另存为

插件安装

Google Chrome浏览器安装方法：打开chrome://extensions/（依次点击右上角三点按钮，更多工具，扩展程序）页面并把crx文件拖入。

最新版Chrome将crx拖入chrome://extensions/即可，如果提示无法安装，右上角开启开发者模式。

其他Chromium内核浏览器安装方法：直接把crx文件拖进浏览器窗口！

使用方法

没啥可说的，即装即用，开启插件马上就能同步数据。

小福利

除同步浏览器数据同步之外，开启插件还支持Gmail和谷歌系列服务的访问。

神秘代码

提取自本插件，即使不安装插件，或者其他非Chromium内核浏览器也可以使用。

HTTPS www.chrome–helper.cc:21830
