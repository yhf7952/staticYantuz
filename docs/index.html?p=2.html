关于博主

博主生于80年代末，是个典型的工科男。大学计算机网络出身，毕业之后做过网站，当过网管，学过编程，现在也算半个程序猿。

鼓捣网站很多年，喜欢琢磨各种事情。随着30岁的到来，就想认认真真的做一件事情。就建了这个博客，希望能将近几年所学所想传达给更多的人。
