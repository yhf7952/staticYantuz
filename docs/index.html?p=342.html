#SEO#后缀.top.win.co等低价域名可能受搜索引擎歧视

做网站绝大多数流量都是来自于搜索引擎，所以受搜索引擎青睐的网站更容易获得高人气。除了外链建设，站内优化之外，最近发现域名后缀也是影响搜索引擎青睐的重要因素！

什么样的域名容易受歧视

表中展示的大多是新加入的域名后缀，除表中的后缀之外.party，.science也会受歧视。

为什么会岐视

说到底是价格原因，因为首年价格低，所以一些作弊者或者诈骗站，广告站和站群等都会选择此类后缀的域名。

百度说明

此类域名注册成本相对较低，因此成为了站群作弊者的温床，部分站群作弊者注册一个站点作弊后，一旦被发现，马上新换另外一个。因此针对这类域名后缀，百度站长平台建议各位站长们不要使用，可能会出现收录延迟等问题。

站在搜索引擎的角度考虑，虽然一刀切可能会误杀很多正常的网站，但是能杜绝了一大部分的垃圾站。而内容正常的网站，往往需要搜索引擎多次核查确认无问题之后，才会开始收录，也就导致了很多新站建站很久，更新也很勤快，但是不收录的情况。

解决方法

在没有完美的解决垃圾站之前，搜索引擎会对top等域名持保留态度。

要想解决问题，最靠谱的方法就是换一个域名。.com .org .net .cn等大众化域名都可以。

将旧域名用301重定向到新域名上，旧域名的权重也会慢慢转移到新站上。
