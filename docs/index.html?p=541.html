直到返程的时候，发现回家的火车票已经卖完了，然后用到了智某抢票软件。偶然间发现了免手续费抢票套路。不知算不算BUG，但确确实实帮我省了20块钱，于是还是决定分享出来，各位鉴别一下。

工具原料

  • 需要抢票的车次，如Z268
  • 抢票工具，如智某火车票（支持先抢票后付费）
  • 12306帐号及官方APP

实现步骤

  1. 打开智某火车票，按照行程选择好车次座位等。支付方式一定不要选择预付费或支付宝代扣
  2. 设置完成之后自动开始抢票。

    抢到之后由于用了加速包，所以原价244，要支付264元。
  3. 这时候先不要支付，只要打开12306APP，找到“订单查询”-“未完成订单”。
    不出意外的话应该有抢票软件为你抢到的车票了，官方价格当然是244元原价。只要在12306中进行支付即可。
  4. 支付成功之后回到智某火车票，取消抢票订单即可。

原理分析

抢票软件是运用自动化脚本为用户不停的查询余票，发现符合条件的票之后即可替用户下单。

而下单必须在官方网站上进行真实的下单。

绕过手续费的做法只是让抢票软件替自己把车票加入购物车，而结算流程仍然是12306中完成。

后记

之前买票心切，所以没有截图。今天写文章再次打开智某火车票时，发现之前加收的20元手续费原本在设置抢票时是记用户选择的。只不过默认值是20元而已。在第一次抢票的时候并没有注意这个选项，而导致最终多出20元的抢票费用。

官方声称加收的费用是为了更快的为用户抢票，猜测应该是付费用户轮询的速度会更快，从而更快抢到票。

各位如果是紧急情况下，可以用此法免费抢票。岩兔站下次再抢票肯定会老老实实选低速模式，免费抢票。
