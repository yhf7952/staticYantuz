百度不收录？增加全站自动推送链接

百度是国内最大的搜索引擎，所以国内网站做的好不好，关键要看百度排名好不好。要想有好的排名，除了内容优质之外，有一个及时的收录也是尤其重要的。除了百度蜘蛛自动抓取之外，百度也提供了4种主动提交链接的方式。

分别是主动推送，自动推送，sitemap，手动提交。四种推送形式都要在百度站长中心进行操作，如果站长同学还没有注册，可要抓紧时间注册啦。

百度站长中心

之前的百度站长中心，目前整合到了百度资源搜索平台，地址是https://ziyuan.baidu.com/dashboard/index ，打开之后登录自己的百度帐号即可开通。

添加网站

想要管理网站，首先要添加网站

添加之前最重要的一点就是验证网站的所有权。

验证方式有三种，分别是文件验证（下载文件上传至网站空间），HTML标签验证（修改主页源码，增加指定标签），CNAME验证（修改站点DNS，增加指定CNAME记录），选择自己方便的验证方式即可，博主用的文件验证，上传验证至服务器主目录即可。

完成验证之后，即可对网站进行管理，本文主要介绍如何及时向百度推送网站更新。进入链接提交页面即可按四种链接提交进行页面提交。

主动推送

百度提供了数据接口，以Post形式提交链接即可进行新增，修改，和删除链接的操作。URL格式为

http://data.zz.baidu.com/urls?site=yourUrl&token=yourToken

用Post模拟测试一下，向指定链接提交URL地址，即可返回成功提示，这个自动推送形式为实时推送。

自动推送

自动推送是在主动推送基础上的简化版，是用户在访问页面的同时，即触发推送脚本，将当前页面推送至百度。

<script>
(function(){
    var bp = document.createElement('script');
    var curProtocol = window.location.protocol.split(':')[0];
    if (curProtocol === 'https') {
        bp.src = 'https://zz.bdstatic.com/linksubmit/push.js';
    }
    else {
        bp.src = 'http://push.zhanzhang.baidu.com/push.js';
    }
    var s = document.getElementsByTagName("script")[0];
    s.parentNode.insertBefore(bp, s);
})();
</script>

将以上代码添加至网站的每个页面，即可实现自动推送。

sitemap

首先网站要建立一份完整的sitemap。如果您与我一样使用wordpress，可以参见wordpress自动生成站点地图插件Google XML Sitemaps 

准备好sitemap之后，在后台提交即可

手动提交

这个没啥好说的，将需要提交的链接全部复制到输入框，依次提交即可。

结语

链接提交只是一种手段，而不是万能的，有的同学可能问了，我一天提交好几次，为什么百度就是不收录呢？百度的收录会考察很多因素的，比如站点内容，备案情况，是否违规，是否原创，网站规模，时间等等，也有很多不确定因素。如果是新站的话，更要猥琐发育，只要坚持原创，百度肯定会收录的。
