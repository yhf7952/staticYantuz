HTTPS网站百度云加速和阿里云空间避坑指南

2018-02-17 分类：资源分享 阅读(1647) 评论(0)

文章目录

  • 阿里云虚拟空间
  • 百度云加速

由于博客部署于国外，所以国内访问比较慢。由此，博主打算将网站迁至国内，自此开始了阿里云的备案流程，详见“阿里云备案全记录”。历时10几天备案顺利通过了，但是问题也来了，主要是博客先前采用https的原因，导致备案之后的诸事不顺。

备案所有的也就是两件事：

  • 国内服务器
  • 百度云加速

但是两件事都如期实现，现在将备案后的折腾记录至此，希望后来者能少走弯路。

阿里云虚拟空间

服务器迁回国内本来是购买的阿里云空间，详见“体验阿里云6元年付虚拟主机”，因为网站访问量不大，数据量也不大，所以虚拟主机足够用了。

等到备案成功之后，迁移网站-更改解析。才发现阿里云虚拟空间不支持https。检测了端口，443压根就没开通。

联系客服之后得知阿里云只有云服务器ECS才支持https的，而ECS产品最低价也要几百元一年，等我的博客发展壮大之后再转移吧。

百度云加速

既然国内的虚拟主机没法用，还有百度云加速，只要能用国内的CDN那么即使服务器在国外，经过CDN加速，访问速度应该也差不到哪去，但是百度云加速居然也不行。

又是切换DNS指向了百度云加速的地址，加速是生效了，可是在百度云加速看到了这样一段话：

使用HTTPS加速的前提条件。一键申请域名专有的HTTPS证书，仅支持专业版或更高版本免费申请。（开通后24小时内生效）

专业版又要1190元/年。唉，果然是贫穷限制了我的HTTPS进程。

注意：百度云加速新接入网站会对www和@记录做初步访问审核，因网站无法访问， 网站打不开或者没有实际内容的站点我们现在会停止加速服务,仅可使用DNS解析功能，页面会显示已违规。如果要解除已违规状态，需联系客服核实并解除。
