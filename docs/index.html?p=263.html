腾讯云服务器 CVM 自助增开自定义端口

2018-03-10 分类：资源分享 阅读(877) 评论(0)

文章目录

  • 需求
  • 配置安全组

博主之前入手了腾讯云服务器，也把网站迁移到了腾讯云上。云主机建站当然是没总是了，80口443端口和ssh的22端口都是默认开通的，那么如果希望搭建其他非默认端口的服务要怎么操作呢？

需求

比如楼主希望在8888端口上搭建一个AppNode服务器面板应用，搭建完成之后是不能通过http://myip:8888/进行访问的。就是因为腾讯云默认限制了非常用端口。

如果想开放端口，并不是调整Linux防火墙虚拟机防火墙，而只要调整腾讯云服务器安全组即可。

配置安全组

首先进入到云主机管理平台地址：https://console.cloud.tencent.com/cvm/index，在需要增设端口的云主机后点击“更多”-“配置安全组”

在弹出的对话框中点击“新建或查看我的安全组详情”。

也可以直接通过链接https://console.qcloud.com/cvm/securitygroup设置安全组。

进入安全组之后可以新建或直接修改原有默认安全组

点击“修改规则”进入安全组规则修改页面。

规则分为“入站规则”和“出站规则”两块，入站规则主要是开启服务器的端口，而出站是规定服务器能访问哪些地址。所以一般情况下只需要设置“入站规则”即可。

  • 来源：如允许所有IP访问，填写“0.0.0.0/0”即可
  • 协议端口：必须以TCP或UDP开头，如：TCP:8888
  • 策略：默认允许
  • 备注：填写备注信息，如AppNode控制台

配置完成之后访问http://myip:8888/试一下，是否正常访问。
