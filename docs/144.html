#Chrome插件#RSS阅读器Slick RSS

2018-01-30 分类：资源分享 阅读(1731) 评论(0)

文章目录

  • 获取方法
  • 使用方法
  • 注意

RSS(Really Simple Syndication)是一种描述和同步网站内容的格式，是使用最广泛的XML应用。

网络上的信息千千万，而个人关注的信息往往是有限的，怎么从众多的的信息中准确的获取自己需要的内容，就需要有到RSS阅读器了，这里推荐一款基于Chrome的RSS阅读插件Slick RSS。

用RSS阅读器有什么优势呢？

  1. 时效，一般RSS阅读器都会以固定时间间隔进行查询订阅源，有更新时能第一时间通知用户。
  2. 快速，说白了RSS是其于XML的一种应用，XML保证了以最少的体积，传达了最多的内容。
  3. 简洁，RSS阅读器只呈现文章内容，而没有多余的广告或其他无关因素，更能专注于内容。

获取方法

Chrome应用市场：https://chrome.google.com/webstore/detail/ealjoljnibpdkocmldliaoojpgdkcdob

无法访问应用市场的用户可前往https://chrome-extension-downloader.com/输入Chrome市场地址，进行下载，然后将crx文件拖入浏览器完成安装

使用方法

订阅网站

找到你想要订阅的网站，找到网站的RSS源，如我的网站订阅源是https://static.yantuz.cn/feed/并将源添加到订阅列表。设置完成后记得点击右上角“save”保存设置。

简单设置

以图片表示吧，可以设置，自动更新间隔，导入导出等。

其中入导出用的也是标准的XML形式，可以导入任何一款RSS阅读器，也可以由任何一款RSS阅读器进行导入，分享一下博的的几个订阅站点：

<?xml version="1.0" encoding="ISO-8859-1"?><opml version="2.0">
<head><title>Slick RSS OPML Export</title></head>
<body><outline type="rss" text="如有乐享" xmlUrl="http://51.ruyo.net/feed/"/>
<outline type="rss" text="拓扑社" xmlUrl="http://www.tuopushe.com/feed"/>
<outline type="rss" text="分享便宜的美国vps" xmlUrl="http://www.138vps.com/rss.php"/>
<outline type="rss" text="91云(91yun.org)" xmlUrl="https://www.91yun.co/feed"/>
<outline type="rss" text="石家庄论坛-石家庄二手网-石家庄极火的本地社区网站" xmlUrl="http://www.sehand.com/forum.php?mod=rss"/>
<outline type="rss" text="秋水逸冰" xmlUrl="https://teddysun.com/feed"/>
<outline type="rss" text="逗比根据地" xmlUrl="https://doub.io/feed/"/>
<outline type="rss" text="免费资源部落" xmlUrl="https://www.freehao123.com/feed/"/>
<outline type="rss" text="皮皮虾窝" xmlUrl="http://ppx.ink/feed"/>
<outline type="rss" text="岩免站" xmlUrl="https://static.yantuz.cn/feed/"/>
</body>
</opml>

阅读文章

左边选择站点，右边展示站点内容，图文并貌的的，未读文章会加深显示，可以一键标记文章为已读。

注意

虽然是chrome插件，但是这个RSS阅读器的设置和订阅源是不能通过Chrome自动同步的，所以多台电脑可以分别设置不同的订阅源，在家关注娱乐新闻等，在单位关注行业新闻。
